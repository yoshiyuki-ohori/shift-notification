操作マニュアル

共通操作

  1. 月を切り替える
    ヘッダーの ◀ ▶ ボタンで対象月を前後に移動する
  2. データを再読み込みする
    ヘッダー右の「再読み込み」ボタンをクリックする
  3. タブを切り替える
    画面上部の「施設一覧」「労基チェック」「配置充足率」「シフト希望」「配置管理」をクリックする

施設一覧タブ

  1. 施設のシフト一覧を開く
    施設名の行をクリックして展開する。もう一度クリックで閉じる
  2. シフトを確認する
    各日付セルに表示されるチップ（時間帯＋氏名）が配置済みシフト
  3. シフトを追加する（既存シフトデータへの直接追加）
    日付セルの「+」ボタンをクリック → 時間帯を選択 → 職員名を検索して選択 →「保存」をクリック
未配置日（赤）
日曜
土曜

労基チェックタブ

  1. チェック結果を確認する
    タブを開くと自動でチェックが実行される。違反(赤)・警告(黄)がカード形式で一覧表示される
  2. 重大度で絞り込む
    フィルターの「全て / 違反のみ / 警告のみ」プルダウンを切り替える
  3. 特定の職員に絞り込む
    「職員名で絞り込み」テキストボックスに名前を入力する

配置充足率タブ

  1. エリア別サマリーを確認する
    上部カードにエリア別の不足/注意/OK件数と平均充足率が表示される
  2. 施設ごとの不足スロットを確認する
    施設名をクリックして展開する。赤=不足、黄=注意のセルが表示される
不足（最低人数未満）
注意（最低人数は確保、推奨人数未満）
OK（推奨人数以上）
前提: スプレッドシートの「必要配置」シートに施設ID・時間帯・曜日種別ごとの最低/推奨人数が登録されている必要があります

シフト希望タブ

  1. 提出状況を確認する
    上部カードに提出済み人数・未提出人数・出勤可能日の合計が表示される。未提出者の名前はカード下に一覧表示される
  2. 日付×職員グリッドを確認する
    横軸が日付（1日～末日）、縦軸が提出済み職員名。出勤可能日は緑背景で表示される
  3. 希望を採用して仮配置に反映する
    出勤可能セルのチェックボックスをONにして時間帯・施設を編集 →「仮配置に反映」ボタンで一括送信
  4. 希望なし日に職員を追加する（カンバン）
    画面下部の「従業員プール」から名前カードをクリック → グリッド上部の日付ヘッダーをクリック → その日に行が追加される
出勤可能日
合計人数が少ない日
カンバンで追加された行

配置管理タブ - 全体の流れ

  1. 「配置管理」タブを開く
    全施設の日付×時間帯グリッドが一覧表示される。ツールバーに配置済数・不足数・充足率が表示される
  2. 各施設の日別スロットに職員を配置していく
    （下記「職員を配置する」を参照）
  3. 全施設の配置が完了したら「確定」ボタンで確定する
    （下記「仮配置を確定する」を参照）

配置管理 - 職員を配置する

  1. 施設を展開する
    施設名の行をクリックして日付グリッドを表示する
  2. 配置したいスロットの「+ 追加」ボタンをクリック
    該当日付・時間帯の「+ 追加」ボタンを押す。職員選択モーダルが開く
  3. 候補者一覧から職員を選ぶ
    一覧は上から順に「希望(緑)→どちらでも(黄)→希望なし(灰)→NG(赤)」で並んでいる。同カテゴリ内は月間配置数が少ない順。ラジオボタンをクリックして選択する
  4. 「配置する」ボタンをクリック
    配置が保存される。同日同時間帯に同じ職員が別施設にも配置されている場合は赤色の重複アラートが画面上部に表示される
NG の職員を選択した場合、確認ダイアログが表示されます。意図的でなければキャンセルしてください

配置管理 - 候補者を絞り込む

  1. 検索ボックスに名前または社員番号を入力する
    モーダル上部の検索欄に入力すると候補者がリアルタイムで絞り込まれる

配置管理 - 配置を削除する

  1. 削除したい職員チップの × ボタンをクリック
    確認ダイアログで「OK」を押すと配置が削除される

配置管理 - 仮配置を確定する

  1. 重複アラートがないことを確認する
    同じ職員が同日同時間帯に複数施設にいると確定できません。チップの × で重複を解消してください
  2. ツールバーの「確定」ボタンをクリック
    確認ダイアログで「OK」を押す
  3. 結果を確認する
    成功すると「確定完了: ○件」と表示される。シフトデータシートにコピーされ、既存のLINE通知・LIFF表示に反映される
確定前に労基チェックが自動実行されます。違反がある場合は結果に表示されるので「労基チェック」タブで詳細を確認してください

配置管理 - 仮配置をクリアする

  1. ツールバーの「クリア」ボタンをクリック
    確認ダイアログで「OK」を押す。対象月の仮配置データが全て削除される
クリアすると配置データは元に戻せません。確定済みデータ（シフトデータシート）には影響しません

配置管理 - 色の見方

スロット背景 赤 = 最低人数に不足
スロット背景 黄 = 最低人数は確保、推奨人数に不足
スロット背景 緑 = 推奨人数以上
チップ左線 緑 = 希望一致
チップ左線 黄 = どちらでも
チップ左線 赤 = NG（要注意）
チップ左線 灰 = 希望なし

スプレッドシート メニュー操作

  1. 配置管理 > 仮配置を確定
    スプレッドシートのメニューからも確定が可能。重複チェック→労基チェック→コピー→LINE通知の順で実行される
  2. 配置管理 > 仮配置をクリア
    対象月の仮配置を全削除する
  3. 配置管理 > 仮配置シート表示
    仮配置シートに直接ジャンプしてデータを確認する

月次シフト配置の運用フロー

  1. 希望・出勤メニュー > 希望収集を開始
    対象月と締切日を入力。職員にLINE通知が送信され、各自がシフト希望を提出する
  2. 希望提出を待つ
    締切2日前に未提出者へ自動リマインド。手動で「未提出者にリマインド送信」も可能
  3. 希望・出勤メニュー > 希望収集を締切
    提出を締め切る
  4. ダッシュボード「配置管理」タブで配置作業
    全施設の各スロットに職員を割り当てる。希望者が上位表示されるので効率よく配置できる
  5. 「配置充足率」タブで不足を確認
    不足スロットがないか最終チェック
  6. 「労基チェック」タブで違反を確認
    連勤超過や時間超過がないかチェック
  7. 「配置管理」タブ > 確定ボタン
    シフトデータシートにコピーされる
  8. シフト管理メニュー > 一括送信（本番）
    確定シフトが全職員のLINEに通知される。LIFF マイシフト画面にも自動反映
