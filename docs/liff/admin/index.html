    еҗҢгҒҳиҒ·е“ЎгҒҢеҗҢж—ҘеҗҢжҷӮй–“еёҜгҒ«иӨҮж•°ж–ҪиЁӯгҒ«гҒ„гӮӢгҒЁзўәе®ҡгҒ§гҒҚгҒҫгҒӣгӮ“гҖӮгғҒгғғгғ—гҒ® × гҒ§йҮҚиӨҮгӮ’и§Јж¶ҲгҒ—гҒҰгҒҸгҒ гҒ•гҒ„
  2. гғ„гғјгғ«гғҗгғјгҒ®гҖҢзўәе®ҡгҖҚгғңгӮҝгғігӮ’гӮҜгғӘгғғгӮҜ
    зўәиӘҚгғҖгӮӨгӮўгғӯгӮ°гҒ§гҖҢOKгҖҚгӮ’жҠјгҒҷ
  3. зөҗжһңгӮ’зўәиӘҚгҒҷгӮӢ
    жҲҗеҠҹгҒҷгӮӢгҒЁгҖҢзўәе®ҡе®ҢдәҶ: в—Ӣд»¶гҖҚгҒЁиЎЁзӨәгҒ•гӮҢгӮӢгҖӮгӮ·гғ•гғҲгғҮгғјгӮҝгӮ·гғјгғҲгҒ«гӮігғ”гғјгҒ•гӮҢгҖҒж—ўеӯҳгҒ®LINEйҖҡзҹҘгғ»LIFFиЎЁзӨәгҒ«еҸҚжҳ гҒ•гӮҢгӮӢ

  1. гғ„гғјгғ«гғҗгғјгҒ®гҖҢгӮҜгғӘгӮўгҖҚгғңгӮҝгғігӮ’гӮҜгғӘгғғгӮҜ
    зўәиӘҚгғҖгӮӨгӮўгғӯгӮ°гҒ§гҖҢOKгҖҚгӮ’жҠјгҒҷгҖӮеҜҫиұЎжңҲгҒ®д»®й…ҚзҪ®гғҮгғјгӮҝгҒҢе…ЁгҒҰеүҠйҷӨгҒ•гӮҢгӮӢ

й…ҚзҪ®з®ЎзҗҶ - иүІгҒ®иҰӢж–№

гӮ№гғӯгғғгғҲиғҢжҷҜ иөӨ = жңҖдҪҺдәәж•°гҒ«дёҚи¶і
гӮ№гғӯгғғгғҲиғҢжҷҜ й»„ = жңҖдҪҺдәәж•°гҒҜзўәдҝқгҖҒжҺЁеҘЁдәәж•°гҒ«дёҚи¶і
гӮ№гғӯгғғгғҲиғҢжҷҜ з·‘ = жҺЁеҘЁдәәж•°д»ҘдёҠ
гғҒгғғгғ—е·Ұз·ҡ з·‘ = еёҢжңӣдёҖиҮҙ
гғҒгғғгғ—е·Ұз·ҡ й»„ = гҒ©гҒЎгӮүгҒ§гӮӮ
гғҒгғғгғ—е·Ұз·ҡ иөӨ = NGпјҲиҰҒжіЁж„Ҹпјү
гғҒгғғгғ—е·Ұз·ҡ зҒ° = еёҢжңӣгҒӘгҒ—

гӮ№гғ—гғ¬гғғгғүгӮ·гғјгғҲ гғЎгғӢгғҘгғјж“ҚдҪң

  1. й…ҚзҪ®з®ЎзҗҶ > д»®й…ҚзҪ®гӮ’зўәе®ҡ
    гӮ№гғ—гғ¬гғғгғүгӮ·гғјгғҲгҒ®гғЎгғӢгғҘгғјгҒӢгӮүгӮӮзўәе®ҡгҒҢеҸҜиғҪгҖӮйҮҚиӨҮгғҒгӮ§гғғгӮҜвҶ’еҠҙеҹәгғҒгӮ§гғғгӮҜвҶ’гӮігғ”гғјвҶ’LINEйҖҡзҹҘгҒ®й ҶгҒ§е®ҹиЎҢгҒ•гӮҢгӮӢ
  2. й…ҚзҪ®з®ЎзҗҶ > д»®й…ҚзҪ®гӮ’гӮҜгғӘгӮў
    еҜҫиұЎжңҲгҒ®д»®й…ҚзҪ®гӮ’е…ЁеүҠйҷӨгҒҷгӮӢ
  3. й…ҚзҪ®з®ЎзҗҶ > д»®й…ҚзҪ®гӮ·гғјгғҲиЎЁзӨә
    д»®й…ҚзҪ®гӮ·гғјгғҲгҒ«зӣҙжҺҘгӮёгғЈгғігғ—гҒ—гҒҰгғҮгғјгӮҝгӮ’зўәиӘҚгҒҷгӮӢ

жңҲж¬ЎгӮ·гғ•гғҲй…ҚзҪ®гҒ®йҒӢз”Ёгғ•гғӯгғј

  1. еёҢжңӣгғ»еҮәеӢӨгғЎгғӢгғҘгғј > еёҢжңӣеҸҺйӣҶгӮ’й–Ӣе§Ӣ
    еҜҫиұЎжңҲгҒЁз· еҲҮж—ҘгӮ’е…ҘеҠӣгҖӮиҒ·е“ЎгҒ«LINEйҖҡзҹҘгҒҢйҖҒдҝЎгҒ•гӮҢгҖҒеҗ„иҮӘгҒҢгӮ·гғ•гғҲеёҢжңӣгӮ’жҸҗеҮәгҒҷгӮӢ
  2. еёҢжңӣжҸҗеҮәгӮ’еҫ…гҒӨ
    з· еҲҮ2ж—ҘеүҚгҒ«жңӘжҸҗеҮәиҖ…гҒёиҮӘеӢ•гғӘгғһгӮӨгғігғүгҖӮжүӢеӢ•гҒ§гҖҢжңӘжҸҗеҮәиҖ…гҒ«гғӘгғһгӮӨгғігғүйҖҒдҝЎгҖҚгӮӮеҸҜиғҪ
  3. еёҢжңӣгғ»еҮәеӢӨгғЎгғӢгғҘгғј > еёҢжңӣеҸҺйӣҶгӮ’з· еҲҮ
    жҸҗеҮәгӮ’з· гӮҒеҲҮгӮӢ
  4. гғҖгғғгӮ·гғҘгғңгғјгғүгҖҢй…ҚзҪ®з®ЎзҗҶгҖҚгӮҝгғ–гҒ§й…ҚзҪ®дҪңжҘӯ
    е…Ёж–ҪиЁӯгҒ®еҗ„гӮ№гғӯгғғгғҲгҒ«иҒ·е“ЎгӮ’еүІгӮҠеҪ“гҒҰгӮӢгҖӮеёҢжңӣиҖ…гҒҢдёҠдҪҚиЎЁзӨәгҒ•гӮҢгӮӢгҒ®гҒ§еҠ№зҺҮгӮҲгҒҸй…ҚзҪ®гҒ§гҒҚгӮӢ
  5. гҖҢй…ҚзҪ®е……и¶ізҺҮгҖҚгӮҝгғ–гҒ§дёҚи¶ігӮ’зўәиӘҚ
    дёҚи¶ігӮ№гғӯгғғгғҲгҒҢгҒӘгҒ„гҒӢжңҖзөӮгғҒгӮ§гғғгӮҜ

    гӮ·гғ•гғҲгғҮгғјгӮҝгӮ·гғјгғҲгҒ«гӮігғ”гғјгҒ•гӮҢгӮӢ
